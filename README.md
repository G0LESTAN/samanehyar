# سامانه‌یار اداری فرهنگیان

---

## توضیح کوتاه

سامانه‌یار اداری فرهنگیان افزونه‌ای سبک و کاربردی برای دسترسی سریع به بخش‌های اصلی سامانه اداری است. با این ابزار می‌توانید به‌سادگی وارد سامانه شوید و از آخرین اخبار و بخشنامه‌های آموزش‌وپرورش مطلع بمانید. اعلان‌های جدید به‌صورت خودکار در مرورگر نمایش داده می‌شوند.

**Short description (EN):**
Samanehyar is a lightweight, practical browser extension for quick access to the main sections of the administrative portal. It helps users log in quickly and receive the latest news and directives as browser notifications.

## ویژگی‌ها / Features

* دسترسی سریع به بخش‌های اصلی سامانه اداری
* ورود سریع (یک کلیک به صفحه ورود)
* اطلاع‌رسانی خودکار (push notifications در مرورگر)
* طراحی سبک و کم‌حجم برای عملکرد سریع
* قابل نصب در Chrome و Firefox

**EN — Features:**

* Quick access to main portal sections
* One-click login shortcut
* Automatic browser notifications for news/directives
* Lightweight and fast
* Works on Chrome & Firefox

---

## نحوه استفاده / Usage

1. پس از نصب آیکون افزونه کنار نوار آدرس مرورگر نمایش داده می‌شود.
2. روی آیکون کلیک کنید تا منوی اصلی باز شود — شامل لینک‌های دسترسی سریع، وضعیت اعلان‌ها و دکمه ورود است.
3. اعلان‌ها به‌صورت push در مرورگر نمایش داده می‌شوند (در صورت فعال بودن مجوزهای اعلان).

**EN — Usage:**

* Click the extension icon to open the quick-access menu.
* Use the login shortcut to sign in to the portal.
* Enable notifications when prompted to receive automatic news/directive alerts.
